# rp-game

Two-player visual RPG battle (tkinter).

## Run

- `python main.py`

## Default controls

- Player 1: `A` = Attack | `S` = Check HP | `D` = Flee
- Player 2: `竊秦 = Attack | `竊伝 = Check HP | `竊蛋 = Flee

## Custom keybindings (optional)

- Copy `controls.example.json` to `controls.json` and edit the keys.
